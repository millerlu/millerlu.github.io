Gmail垃圾郵件無法讓Outlook收信的解決方法

發表於 2016-06-11   |   分類於 網路科技

許多公司把Gmail當成內部與對外的聯絡信箱，並附掛專屬網域申請（企業版）；但還是將Gmail設定於Outlook信箱收信，而最常遇到的郵件（包含垃圾郵件）無法被本機端電腦收信，可是工作最怕是收件漏信，所以秉持著寧可錯殺不可放過的精神，紀錄設定的方式給大家參考看看。
（其實是：迷之聲！同事：為什麼我都沒有收到信阿！)
（吼…..很爛耶！)
（為什麼我寄的信件變成垃圾阿！＊＠＄＃％～)

設定

登入Gmail後，在右手邊點擊 齒輪小圖 裡，選擇設定

篩選器和封鎖的地址

在設定選項裡，選擇 篩選器 （新版名稱：篩選器和封鎖的地址）
=> 建立新篩選器

包含字詞

包含字詞 => 輸入is:spam

輸入完後 => 在右下角按 繼續

出現彈出視窗( 提醒你篩選的條件是否要再繼續 ) => 確定

篩選的項目

勾選 [v] 不要將它傳送到垃圾郵件

可勾選，將篩選器同時套用到n個相符的會話群組，完成後直接歸類
